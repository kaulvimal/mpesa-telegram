# mpesa-telegram-bot


## Installation

1. Make sure you've installed all requirements
2. Clone this repository:
  `git clone https://github.com/vimalkaul/mpesa-telegram`
3. Install the project using `npm`:
  `npm install`


## Requirements / Dependencies

* NodeJS
* Telegram Bot API token
